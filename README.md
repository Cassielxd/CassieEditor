# cool editor


- [English md](https://github.com/Cassielxd/CassieEditor/blob/main/README-EN.md)
#### 介绍

基于prosemirror和tiptap开发的富文本编辑器项目
prosemirror提供最基础的脚手架
tiptap二次api封装 可以有很大的想象空间
本项目主要用于 电子病历的预研

软件架构说明

2. 前端框架 vue3
